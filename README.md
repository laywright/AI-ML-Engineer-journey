# Student Performance Tracker

## Overview
The Student Performance Tracker is a Python-based project designed to manage and analyze student scores. It allows you to:
- Ingest student data from JSON files or manual input.
- Validate data to ensure it meets structural and numeric requirements.
- Calculate averages and identify top performers.
- Flatten nested score data for ML-ready usage.
- Log invalid or skipped entries for transparency and debugging.

This project is modular, production-ready, and suitable for education analytics pipelines.

---

## Features
- **Data Input**: JSON files + manual input.
- **Validation**: Ensures all profiles are complete and numeric.
- **Analytics**: Computes average scores, identifies top students.
- **Transformation**: Flattens nested dictionaries for machine learning.
- **Logging**: Tracks skipped profiles and errors using Python's logging module.
